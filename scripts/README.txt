start_couchdb.sh - start CouchDB database using couchdb.ini as configuration file
start_nginx.sh - start nginx web-server using nginx.conf as configuration file
start_dev_env.sh - start both nginx and CouchDB. Terminates them both on Ctrl+C

If you copy your config files from samples then by default nginx listens port 7070,
and CouchDB - 7000. 
URL for CouchDB UI:
http://localhost:7000/_utils/
